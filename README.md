# JobBoard 🚀

A full-stack job board web application that allows users to search, filter, and save remote jobs. Built with React, Node.js, Express, and MongoDB.

## Features

- 🔍 Search and filter jobs by title, category, contract type
- 💾 Save and manage jobs in personal dashboard
- 🔐 JWT-based authentication (register/login)
- 👮 Role-based access control (admin/user)
- 📱 Fully responsive on mobile and desktop
- ⚡ Powered by Remotive API (real remote jobs)
- 🛠️ Admin panel to manage users and roles

## Tech Stack

**Frontend**
- React 18 + Vite
- Tailwind CSS
- React Router v6
- TanStack React Query
- Axios

**Backend**
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- bcryptjs (password hashing)
- Remotive Jobs API
